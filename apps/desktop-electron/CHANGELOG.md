# Changelog

## [1.7.3] - 2026-03-06

### Code Quality

- **DNS Leak Test:** Заменён `https-proxy-agent` + `@ts-ignore` на нативный `undici.ProxyAgent` + `undici.fetch` — полная типобезопасность proxy-запросов.
- **Тесты:** Устранены 7× `noExplicitAny` — доступ к private-методам через типизированный интерфейс `ManagerInternals`.
- **JSON responses:** Добавлены type assertions (`as { ip?: string }`) для всех `json()` вызовов.

## [3.0.0] - 2026-03-03

### Крупное обновление дизайна (UI/UX Overhaul v3)

- **Полностью новый визуал:** Ярко оранжевый (#FF4D00) и глубокий черный цвета. Полностью переработана концепция интерфейса на более современный, "дорогой" и технологичный.
- **Новый логотип:** В приложении используется новая иконка "Inferno Bolt" (острый щит + ⚡).
- **Dashboard v3 (Clean Power):**
  - Центральная мультифункциональная кнопка Power с ярким градиентом, подсветкой и гласс-преломлением.
  - Интерактивный DepthBackground (эффект глубины через 3D сетку и плавающие orbs с параллаксом от мыши).
  - Карточки информации (InfoCard) с glassmorphism, эффектом lift при наведении и аккуратным свечением.
  - Круговые графики `SpeedGraph` с утолщенными линиями, 3-stop градиентом и SVG свечением.
  - Увеличенные на 10% шрифты и цифры для лучшей читаемости.
- **Единый стиль всех элементов:**
  - Оранжевые градиентные action-кнопки по всему приложению (AddServerModal, ServerItem, ServerList, Onboarding) с подсветкой и "стеклянными" бликами.
  - Единые анимации `whileHover` (плавное масштабирование) и `whileTap` через Framer Motion. Убраны старые "плавающие" (дешевые) анимации на BottomNav.
- **Окно и навигация:**
  - TitleBar очищен от иконки, оставлено только название приложения. Элементы управления окном (свернуть/закрыть) увеличены до 46x40px (стандарт Windows), иконки стали толще и контрастнее.
  - BottomNav ("Perfect Dock"): Иконки в навигационной панели увеличены на 20%, тексты крупнее, добавлены градиентные подложки для активных вкладок.
- **Onboarding:** Свежий экран приветствия, яркие премиальные оранжевые партиклы, новые градиентные кнопки настройки.
- **Оптимизация:** Значительно уменьшено использование GSAP (основная миграция на Framer Motion для микро-взаимодействий). Исправлены баги отображения текстов.
