## Что это

**EgoistShield** — desktop-приложение для Windows 10/11, которое объединяет:

- импорт и локальное управление сетевыми профилями и рабочими конфигурациями;
- выбор рабочего сетевого сценария и запуск через встроенные сетевые компоненты;
- системные функции клиента: автозапуск, автоматическое восстановление рабочего состояния, ручная проверка обновлений и служебные сетевые инструменты;
- отдельный экран управления системным DNS;
- журналы, диагностику и метрики приложения.

Проект ориентирован на пользователей, которым нужен **локально управляемый Windows-сетевой хаб** с современным UI и воспроизводимым релизным контуром.

## Что нового в 4.0.11

- **Скрытый фоновый сетевой модуль**: дополнительный runtime больше не всплывает как отдельное окно или консоль и работает внутри EgoistShield.
- **Автоматическое самовосстановление managed runtime**: старые фоновые сборки корректно заменяются встроенной скрытой версией при запуске приложения.
- **Обновлённый release-контур 4.x**: installer, `latest.yml` и проверка release-артефактов синхронизированы для GitHub Release канала.
- **Укреплённый upgrade-path**: обновление поверх предыдущих desktop-версий стало надёжнее и чище.
- **Системный DNS и сетевые сценарии**: публичные описания и документация выровнены под позиционирование приложения как сетевого хаба.
- **Оптимизация production-shell**: скрытые runtime-компоненты, обновления и desktop smoke-проверки сведены в единый воспроизводимый процесс.

Подробности по изменениям: [CHANGELOG.md](CHANGELOG.md).

## Интерфейс

- **Dashboard** с крупной кнопкой состояния, карточками узла и сетевыми метриками.
- **Server Center** с 3D-глобусом, списком узлов, фильтрацией и подписками.
- **DNS Center** для глобальной Windows DNS-конфигурации.
- **Отдельный сервисный экран** для профилей, служебных инструментов, диагностики и cache-cleanup.
- **Settings** для системных переключателей приложения, обновлений и входных точек в DNS и сервисные инструменты.

## Быстрый старт

1. Скачайте установщик из [последнего релиза](https://github.com/egoistgorbachev/egoistshield/releases/latest).
2. Установите приложение на Windows 10/11 x64.
3. Добавьте сетевые профили через `Ctrl+V`, импорт файла или ссылку конфигурации.
4. Выберите рабочий сценарий вручную или используйте `Smart Connect`.
5. При необходимости настройте системный DNS на отдельном экране `DNS Center`.
6. Если нужны сервисные утилиты или системная диагностика, откройте отдельный сервисный экран.

## Публичное позиционирование и добросовестное использование

Публичное описание проекта и его позиционирование приведены в консервативной форме с учётом действующих требований российского законодательства на начало апреля 2026 года.

- Проект описывается как **сетевой хаб для настройки подключений, DNS и локальных сетевых сценариев**, а не как средство для обхода ограничений доступа.
- Приложение предполагает использование с собственной инфраструктурой пользователя или с конфигурациями, на использование которых у пользователя есть законные основания.
- Пользователь и оператор собственной инфраструктуры самостоятельно оценивают применимость требований к обработке данных и локальным сетевым настройкам.
- Если вы добавляете телеметрию, учётные записи или внешние backend-сервисы, отдельно оцените требования законодательства о персональных данных и уведомительном контуре.

Это не юридическая консультация, а продуктовый и редакционный guardrail для публичной страницы проекта.
