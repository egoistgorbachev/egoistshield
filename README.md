## 📥 Установка

1. **Скачай** → [**EgoistShield_Setup.exe**](https://github.com/egoistgorbachev/egoistshield/releases/latest)
2. **Запусти** скачанный файл — установка автоматическая

> 💡 **Обновления приходят автоматически** — приложение само уведомит о новой версии и установит обновление в один клик.

## 🎨 Интерфейс

- 🌑 **Тёмный дизайн** — стильный тёмный интерфейс с плавными анимациями
- 🟢 **Кнопка подключения** — большая кнопка с визуальной индикацией статуса (зелёная = защищён, оранжевая = отключён)
- 🌍 **3D-глобус** — интерактивная карта мира с точками серверов, пингом и флагами стран
- 📈 **Графики трафика** — приём/отдача в реальном времени с визуализацией
- 📋 **Управление подписками** — просмотр лимита, использованного трафика и оставшихся дней

---

## 🔒 Безопасность

- Приложение **не собирает** никаких персональных данных
- Весь трафик шифруется протоколами VPN
- Код проходит регулярный внутренний аудит безопасности
- При удалении — все настройки полностью очищаются

## ❓ FAQ

<details>
<summary><b>Как добавить сервер?</b></summary>

Скопируй VPN-ссылку и нажми **Ctrl+V** в приложении — сервер добавится автоматически. Также можно добавить подписку по URL.
</details>

<details>
<summary><b>Как обновить?</b></summary>

Обновления приходят автоматически. При наличии новой версии появится уведомление с кнопкой «Установить и перезапустить».
</details>

<details>
<summary><b>Как удалить?</b></summary>

Через «Установку и удаление программ» в Windows → EgoistShield → Удалить. Все настройки будут очищены автоматически.
</details>
